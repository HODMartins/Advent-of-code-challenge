# Advent of Code 2024 - Day 2: Red-Nosed Reports

This repository contains my Python solution for **Advent of Code 2024 - Day 2**.

The challenge focuses on analyzing reactor safety reports and determining whether each report follows valid safety rules.

## Problem Summary

Each report consists of a sequence of numbers called **levels**.

A report is considered **safe** if:

- The levels are either all increasing or all decreasing
- The difference between any two adjacent levels is at least **1** and at most **3**
